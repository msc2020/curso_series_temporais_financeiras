Questo 1

Suponha que a srie diria de log-retorno de um ativo siga o seguinte modelo: \(y_t = 0.01 + 0.2\,y_{t2} + \epsilon_t\) , onde \(\epsilon_t\) um rudo branco Gaussiano com mdia zero e varincia \(0.02\). (a) Determine a mdia e a varincia da srie de retornos \(y_t\). (b) Calcule as autocorrelaes de ordem 1 e 2 de \(y_t\). (c) Assuma que o valor de \(y\) em \(t = 100\), \(y_{100}\), seja \(-0.01\) e o valor de \(y\) em \(t = 99\), \(y_{99}\), seja \(0.02\). Calcule a previso um passo frente da srie de retornos a partir da origem \(t = 100\). Calcule tambm a previso dois passos frente a partir de \(t = 100\).

Resposta 1

Para simplificar as manipulaes algbricas, iremos usar a seguine notao para a srie \(y_t\) definida neste execcio: \[\begin{equation} y_t = 0.01 + 0.2\,y_{t-2}+\epsilon_t = \alpha + \phi\,y_{t-2} + \epsilon_t, \end{equation}\] onde \(\alpha = 0.01, \phi=0.2\) e \(\epsilon_t\) um rudo branco com mdia zero e varincia \(\sigma^2 = 0.02\).

(a) Clculo da mdia \(E(y_t)\):

Usando as propriedades do valor esperado \(E\), a mdia de \(y_t\) dada por \[\begin{eqnarray} E\{y_t\} &=& E\{\alpha + \phi_2\,y_{t-2} + \epsilon_t\}\\ &=& E\{\alpha\} + \phi_2\, E\{y_{t-2}\} + E\{\epsilon_t\}, \ \ \text{pela linearidade de E}\\ &=& \alpha + \phi_2\, E\{y_{t-2}\} + 0, \ \ \text{pois $\epsilon_t$ possui mdia zero} \end{eqnarray}\] Usando a estacionaridade de \(y_t\), temos que \(E\{y_t\} = E\{y_{t-2}\} = \mu\), onde \(\mu\) uma constante. Substituindo na expresso acima temos \[\begin{equation} \mu = \alpha + \phi_2\, \mu \end{equation}\] Logo, a mdia \(\mu\) de \(y_t\) \[\begin{eqnarray} \mu = \frac{\alpha}{1-\phi} = \frac{0.01}{1-0.02} = \frac{0.01}{0.8} = 0.0125. \end{eqnarray}\] Portanto, \(\mu= 0.0125\).

Clculo da varincia \(var(y_t)\): \[\begin{eqnarray} var(y_t) &=& var(\alpha + \phi\,y_{t-2} + \epsilon_t), \ \ \text{usando da definio de $y_t$}\\ &=& \phi^2\,var(y_{t-2}) + var(\epsilon_t) + 2\, cov(y_t-2, \epsilon_t), \ \ \text{pelas propriedades da varincia} \\ &=& \phi^2\,var(y_{t-2}) + \delta^2 + 0, \ \ \text{pois $var(\epsilon_t) = \delta^2$ e $y_t$ e $\epsilon_t$ no so correlacionados} \end{eqnarray}\] Logo, \[\begin{equation} var(y_t) = \frac{\delta^2}{1 - \phi^2}. \end{equation}\] Subsituindo os valores das constantes na expresso acima, temos que \[\begin{eqnarray} var(y_t) &=& \frac{0.02}{1-(0.2)^2} = \frac{0.02}{1-4 * 10^{-2}} \\ &=& \frac{0.02}{0.96} = 0.021 \end{eqnarray}\] Portanto, \(var(y_t) = 0.021\). Como \(var(y_t) = cov(y_t, y_t) = \gamma_0\), ento \(\gamma_0 = 0.021\).

(b) Inicialmente, vamos calcular uma expresso para a autocovarincia de lag \(k\), denotada por \(\gamma_k\): \[\begin{eqnarray} \gamma_k &=& cov(y_t, y_{t-k}) \\ &=& E(y_t\,y_{t-k}) - \mu^2, \ \ \text{pois $E(y_t) = E(y_{t-k}) = \mu=$'mdia constante de $y_t$'}\\ &=& E[(\alpha + \phi\,y_{t-2} + \epsilon_t)\,y_{t-k}] -\mu^2, \ \ \text{j que $y_t = \alpha + \phi\,y_{t-2} + \epsilon_t$} \\ &=& E(\alpha\,y_{t-k} + \phi\,y_{t-2}y_{t-k} + \epsilon_ty_{t-k}) - \mu^2, \ \ \ \text{aps multiplicarmos}\\ &=& \alpha\, E(y_{t-k}) + \phi\,E(y_{t-2}y_{t-k}) + E(\epsilon_ty_{t-k}) - \mu^2, \text{pois $E(aX+bY) = aE(X)+bE(Y)$, $\forall a,b$ constantes e $\forall X, Y$ variveis aleatrias}\\ &=& \alpha\,\mu + \phi\,E(y_{t-2}y_{t-k}) + 0 - \mu^2, \ \ \text{pois srie $y_t$ e o erro $\epsilon_t$ no so correlacionados} \\ &=& \alpha\,\mu + \phi\,E(y_{t-2}y_{t-k}) - \mu^2 + \phi\,\mu^2 - \phi\,\mu^2, \ \ \text{aps somar e subtrair um termo que desejamos}\\ &=& \alpha\,\mu + \phi\,[E(y_{t-2}y_{t-k}) - \mu^2] - \mu^2 + \phi\,\mu^2, \ \ \text{ agrupando os termos}\\ &=& \alpha\,\mu + \phi\,\gamma_{k-2} - \mu^2 + \phi\,\mu^2, \ \ \ \text{pois $E(y_{t-2}y_{t-k}) - \mu^2 = \gamma_{k-2}$ (adiante verificaremos essa identidade)} \end{eqnarray}\] Portanto, \[\begin{equation} \gamma_k = \alpha\,\mu + \phi\,\gamma_{k-2} - \mu^2 + \phi\,\mu^2. (*) \end{equation}\]

Usando \(k=1\) na expresso acima, temos \[\begin{equation} \gamma_1 = \alpha\,\mu + \phi\,\gamma_{-1} - \mu^2 + \phi\,\mu^2. \end{equation}\] Como \(y_t\) estacionria, vale que \(\gamma_k = \gamma_{-k}\). Ento, \[\begin{equation} \gamma_1 = \alpha\,\mu + \phi\,\gamma_{1} - \mu^2 + \phi\,\mu^2 = \alpha\,\mu + \phi\,\gamma_{1} - \mu^2(1- \phi). \end{equation}\] Isolando \(\gamma_1\), ficamos com \[\begin{equation} \gamma_1 = \frac{\alpha\,\mu}{1-\phi} -\mu^2 = 0. \end{equation}\] Ento, substituindo os valores das constantes \(\alpha, \mu, \phi\), a correlao de lag 1 : \[\begin{equation} \rho_1 = \frac{\gamma_1}{\gamma_0} = \frac{0}{\gamma_0} = 0. \end{equation}\] Portanto, a autocorrelao de ordem 1 \(\rho_1 = 0\).

Usando (*), obtemos a autocovarincia de ordem 2, \(\gamma_2\): \[\begin{eqnarray} \gamma_2 &=& \alpha\,\mu + \phi\,\gamma_0 + \mu^2\,(\phi-1)\\ &=& 0.01*0.0125 + 0.2*0.021 + (0.0125)^2*(0.2 - 1)\\ &=& 1.25*10^{-4} + 4.2*10^{-3} - 1.25*10^{-4}\\ &=& 4.2*10^{-3} \end{eqnarray}\] Logo, a autocorrelao de ordem 2 \[\begin{equation} \rho_2 = \frac{\gamma_2}{\gamma_0} = \frac{4.2*10^{-3}}{0.021} = 0.2. \end{equation}\] Portanto, \(\rho_2 = 0.2\).

Para verificar que \(E(y_{t-2}\,y_{t-k}) - \mu^2 = \gamma_{k-2}\), podemos fazer a substituio de variveis \(t^{'} = t-2\) (ou seja, \(t = t^{'}+2\)), ento \[\begin{eqnarray} E(y_{t-2}\,y_{t-k}) - \mu^2 &=& cov(y_{t-2}, y_{t-k}), \ \ \ \text{pois $y_{t}$ estacionria e tem mdia constante}\\ &=& cov(y_{t^{'}}, y_{(t^{'} + 2) - k}), \ \ \ \text{substituindo $t = t^{'} + 2$}\\ &=& cov(y_{t^{'}}, y_{t^{'} - (k-2)}), \ \ \ \text{agrupando os termos}\\ &=& cov(y_{t}, y_{t - (k-2)}), \ \ \ \text{pois $t$ um rtulo arbitrrio para o subndice e pode ser alterado}\\ &=& \gamma_{k-2} \end{eqnarray}\]

Questo 2

Descreva como as FAC e FACP so teis para identificar um modelo ARMA. Como os critrios de informao podem ser usados na construo de um modelo?

Resposta 2

Um modelo ARMA(p, q) pode ser escrito como: \[\begin{equation} y_t = \alpha + \sum_{j=1}^p \phi_j\, y_{t-j} + \sum_{i=0}^q \theta_i\, \epsilon_{t-i}, \end{equation}\] onde os coeficientes e variveis so conforme vistos em aula. Dessa maneira, o valor atual da srie \(y_t\) para um modelo AR(p,q), pode depender do intercepto (drift) \(\alpha\), dos valores passados da srie \(y_k\) para \(k < t\) e do erro atual \(\epsilon_t\) (assumido como rudo branco). Para medir a correlao entre as variveis atuais e passadas utilizamos a funo de autocorrelao (abreviada por ACF, em ingls) e a funo de autocorrelao parcial (PACF, em ingls). A ACF mede a correlao entre duas variveis (atual e passado) e a PACF mede a correlao entre duas variveis eliminando o efeito de outras variveis (uma forma de se fazer isso, aplicando uma regresso linear).

Conforme vimos ao longo do curso, a ACF de uma srie temporal \(y_t\) formada por \(\rho_1, \rho_2, \rho_k, \dots, \rho_k\) onde \(\rho_k = \frac{cov(y_t,\, y_{t-k})}{var(y_t)} = \frac{\gamma_k}{\gamma_0}\). Alm disso, a PACF dada por \(\phi_1, \phi_2, \dots, \phi_k\) em que \(\phi_k\) satisfaz a equo \(y_t = \phi_1\,y_{t-1} + \phi_2\,y_{t-2} + \dots + \phi_k\,y_{t-k} + v_t\).

Adicionalmente, as propriedades dos processos AR(p) e MA(q) podem ser usados na identificao de um modelo ARMA(p, q). De fato, um processo AR(p) descrito por: (i) uma ACF infinita e declinante; e (ii) uma PACF aproximadamente igual a zero (ou truncada) para lags maiores que p.J um processo MA(q) descrito por: (i) uma ACF aproximadamente igual a zero (ou truncada) para lags maiores que q; e (ii) uma PACF infinita e declinante. Analisando o grfico da ACP e FACP podemos estimar visualmente os valores de p e q num modelo ARMA.

Uma propriedade interessante que MA(\(\infty\)) = AR(1) e MA(1) = AR(\(\infty\)).

Conforme vimos em aula, modelos AR podem ser estimados por Mnimos Quadrados Ordinrios (MQO). J os modelos MA e ARMA so estimados atravs do mtodo da Mxima Verossimilhana (MV), pois os erros que atuam como variveis independentes nesses modelos no so diretamente observveis.

Para usarmos os critrios de informao Akaike (AIC) e Bayesiano (BIC), calculamos o AIC (ou BIC) para \(k = 0, \dots, n\), onde \(n\) para um inteiro positivo. O valor da ordem \(k\) que possui o valor AIC (ou BIC) mnimo indica a ordem \(k\) que deve ser escolhida para termos o melhor modelo.

Questo 3

Seja a FAC e FACP (ACF e PACF em ingls, respectivamente) de uma srie de log-retornos mostrada na pgina seguinte. Qual seria um modelo do tipo ARMA razovel para descrever a dinmica dessa srie? Como voc poderia verificar se o modelo escolhido adequado? NOTA: As FAC e FACP mostradas abaixo, comeam no lag = zero. Ento, o primeiro elemento a autocorrelao de ordem zero, que deve ser desconsiderada para a anlise proposta.

fonte: Homework 2 - Questo 3.

Resposta 3

Olhando para o grfico da PACF notamos um alto valor para o lag = 1, sugerindo que o modelo ARMA possui uma parte AR(1). Enquanto isso o decaimento indicado no grfico da ACF, sugere uma parte MA(\(\infty\)).

Vejamos alguns testes computacionais antes de decidir qual ordem escolheremos para nosso modelo ARMA.

TT <- 250
#?arima.sim
ma <- acf(arima.sim(model=list(ma=seq(from=1, to=10000, by = 1)), n=TT), 
           ylab=expression(Y[t]),
           main=expression(paste("MA(10000) p/ ", theta, "=1")))

ar <- pacf(arima.sim(model=list(ar=c(0.9, 0.001, 0.05)), n=TT), 
           ylab=expression(Y[t]),
           main=expression(paste("AR(3) p/ ", theta, "=(0.9, 0.001, 0.05)")))

Os testes computacionais sugerem que nossa o modelo formado por MA(\(\infty\)) e AR(1). Como MA(\(\infty\)) = AR(1), ento escolhemos o modelo ARMA(1, \(\infty\)) que equivale a um ARMA(1, 0).

Para saber se o modelo ARMA(1, 0) proposto est adequado, podemos analisar os seus resduos. Se os resduos tiverem comportamento de rudo branco, ou seja, mdia zero, varincia constante e ausncia de autocorrelao, ento o modelo proposto foi adequado. Podemos fazer isso atravs da anlise das FAC e FACP dos resduos, alm dos testes de Ljung-Box e Box-Pierce. No caso de haver autocorrelao na srie dos resduos, devemos revisar o modelo proposto.
